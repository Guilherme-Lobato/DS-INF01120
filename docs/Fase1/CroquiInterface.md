# Croquis e Justificativa de Layout - Fase 1

O layout foi pensado para fornecer uma experiência direta, onde a ação de digitar e reproduzir texto está em primeiro plano.

### Justificativa do Layout:
- **Cabeçalho:** Barra superior com o nome do sistema e indicador de status.
- **Área de Entrada (Esquerda):** Campo de texto amplo em destaque, pois a inserção do texto é a ação primária do usuário.
- **Painel de Configurações (Esquerda, abaixo do texto):** Controles numéricos para BPM, instrumento, oitava e volume, para acesso rápido.
- **Botão de Ação:** Botão destacado para "Gerar e Tocar", que muda visualmente para "Parar" durante a reprodução para dar feedback imediato.
- **Painel de Visualização (Direita):** Área visual onde cada caractere é um bloco, destacando o evento atual em tempo real para que o usuário acompanhe a música visualmente.
